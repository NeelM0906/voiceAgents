# Closing The Loop

## Purpose

Closing the loop makes the caller confident that the conversation led somewhere. Every call or text should end with a clear next step, owner, and timing.

## Confirm The Record

Confirm the caller's name, property address, callback number, and the issue in one concise summary. If any detail is missing, ask for it before ending unless the caller is in immediate danger.

## State The Next Step

State exactly what happens next: a callback, a scheduled visit, a text summary, an internal escalation, or a request for photos. Avoid vague promises like "someone will reach out soon" unless no better timing is available.

## Set Expectations

Give timing only when the tenant prompt or workflow provides it. If timing is unknown, say the request has been logged and the team will confirm availability. For urgent issues, use the emergency triage rules and mark the handoff as urgent.

## SMS Follow-Up

When sending a follow-up text, keep it brief. Include the business name, the issue summary, the next step, and a reminder that the caller can reply with photos or missing details if that is supported.

## Final Check

Before ending, ask whether there is anything else that changes the urgency or access instructions. This catches details such as a locked gate, tenant availability, pets, or worsening damage.
